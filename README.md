# DC Motor PID Tuning Comparison

This project compares three PID tuning methods applied to a DC motor model:

- Genetic Algorithm (GA)
- Ziegler–Nichols
- Cohen–Coon

## Objectives
Evaluate controller performance using:

- Rise Time
- Settling Time
- Overshoot
- ISE / IAE / ITAE
- Control Energy

## Methods
MATLAB simulations were used to design and analyze the controllers.

## Results
GA tuning produced the fastest and most stable response, while Ziegler–Nichols showed aggressive oscillatory behavior.

## Tools
- MATLAB
- Control System Toolbox
- Genetic Algorithm Optimization
